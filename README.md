## Aperçu

Portfolio personnel développé avec **React 16** et **styled-components**, présentant mon parcours de développeur fullstack basé à Conakry, Guinée. Le site est responsive, supporte le mode clair/sombre et couvre l'ensemble de mon activité : compétences techniques, projets concrets, certifications Udemy, expériences professionnelles et formations.

---

## Fonctionnalités

- **Design responsive** — adaptation mobile, tablette et desktop
- **Thème clair/sombre** — bascule via un toggle dans la barre de navigation
- **Animations fluides** — react-reveal (Fade) sur toutes les sections
- **SEO** — meta tags dynamiques avec react-helmet
- **Visualisation de CV** — lecteur PDF intégré
- **Modale de certification** — détails complets (instructeur, note, durée, suivi)
- **Galerie projets** — grille avec cartes, liens GitHub et démo live
- **Formulaire de contact** — coordonnées directes et réseaux sociaux

## Personnalisation

Toutes les données sont centralisées dans `src/portfolio.js`.

### Informations personnelles

```js
const greeting = {
  title: "Mohamed Hady Diallo",
  nickname: "scriptseinsei",
  subTitle: "Développeur Fullstack · Je construis des solutions numériques pour l'Afrique.",
  resumeLink: "",
  githubProfile: "https://github.com/hadydieye",
};
```

### Ajouter un projet

Ajoutez un objet à `projects.data` :

```js
{
  id: "mon-projet",
  title: "Mon Projet",
  description: "Description du projet.",
  techStack: ["React", "Node.js", "MongoDB"],
  imagePath: "MonProjet.png",     // placez l'image dans public/projects/
  githubLink: "https://github.com/...",
  liveLink: "https://mon-projet.vercel.app",
}
```

### Ajouter une certification

Ajoutez un objet à `certifications.certifications` :

```js
{
  id: "ma-certif",
  title: "Titre de la certification",
  subtitle: "- Sous-titre",
  logo_path: "image.jpg",         // placez l'image dans public/certifications/
  certificate_link: "",
  alt_name: "Udemy",
  color_code: "#00000099",
  details: {
    instructor: "Nom du formateur",
    rating: "4.5/5 (100 avis)",
    students: "10 000 étudiants",
    duration: "10h · 30 leçons",
    date: "1 janvier 2026",
    topics: ["Sujet 1", "Sujet 2"],
  },
}
```

### Modifier les compétences

Dans `skills.data`, chaque section contient un `title`, des `skills` (listes descriptives) et des `softwareSkills` (icônes). Les icônes utilisent [Iconify](https://icon-sets.iconify.design/).

### Changer le thème

Le toggle en haut à droite de la navigation permet de basculer entre le thème clair (`blueTheme`) et le thème sombre (`materialDarkTheme`). Les thèmes sont définis dans `src/theme.js`. Pour changer le thème par défaut, modifiez :

```js
// src/theme.js
export const lightTheme = blueTheme;   // thème clair par défaut
export const darkTheme = materialDarkTheme;  // thème sombre
```

### Modifier les liens sociaux

```js
const socialMediaLinks = [
  {
```

```
│   │   ├── topButton/        # Bouton retour en haut
│   │   └── ...
│   ├── containers/
│   │   ├── Main.js           # Point d'entrée, routes
│   │   └── ...
│   ├── pages/
│   │   ├── home/             # Page d'accueil
│   │   ├── education/        # Formation
│   │   ├── experience/       # Parcours
│   │   ├── projects/         # Projets
│   │   ├── contact/          # Contact
```
